Project Documentation :

1. Project Overview

This project is an AI-powered chatbot system designed to provide intelligent responses, educational news updates, and internship opportunities. The system integrates multiple technologies including AI APIs and web scraping to deliver real-time and useful data.

2. Key Features

  • AI Chatbot: Generates responses using Sambanova API.
  • Educational News: Fetches latest education-related news.
  • Internship Finder: Scrapes internship listings from Internshala and OK Remote.

3. Technologies Used

  • Backend: Django (Python)
  • Frontend: HTML, CSS, JavaScript
  • Database: dbsqlite3
  • AI API: Sambanova
  • Web Scraping: BeautifulSoup, Requests

4. System Architecture

The system is divided into three main modules:

  • Chat Module: Handles user queries and sends them to Sambanova API.
  • News Module: Retrieves education-based news using APIs or scraping.
  • Internship Module: Scrapes internship data from external platforms.

5. AI Chatbot Module

The chatbot takes user input and processes it through the Sambanova API. It returns AI-generated responses in real-time.

Flow:

  • User inputs query
  • Request sent to backend
  • Backend calls Sambanova API
  • Response displayed to user

6. News Feature

The news module provides the latest updates focused on education. It ensures users stay informed about trends, exams, and opportunities.

Sources:

  • Education news APIs OR
  • Web scraping from trusted news websites

7. Internship Scraper Module

This module collects internship opportunities from:

  • Internshala
  • OK Remote

Functionality:

  • Search internships based on skill
  • Scrape job title, company, location, and link
  • Display results dynamically

Tools Used:

  • requests – to fetch web pages
  • BeautifulSoup – to parse HTML

8. Workflow

  • User interacts with chatbot interface
  • Chooses feature (Chat / News / Internships)
  • Backend processes request
  • Data fetched from API or scraping
  • Results displayed on UI

9. Future Enhancements

  • Add more job platforms (LinkedIn, Indeed)
  • Improve AI response accuracy
  • Improve system

10. Conclusion

This AI Educational Navigator combines artificial intelligence and real-time data extraction to provide a powerful tool for students. It helps users get intelligent answers, stay updated with educational news, and find relevant internship opportunities.
