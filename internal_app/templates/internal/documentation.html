AI-Guided Application Documentation

Overview

This Django project provides a simple full-stack environment with user authentication, an AI chat tutor, education news integration, and an internship finder module.

Features

  • User sign‑up/log‑in using Django auth.
  • AI tutor powered by SanbanoVa API with session history.
  • Education news fetched from GNews API.
  • Internship finder backed by a database model and filterable via GET parameters.
  • Admin dashboard with custom header and models registered.

Internship Finder Logic

The page located at /internships/ is rendered by internal_app.views.internships. Internships are stored in the Internship model. Filters are passed as query parameters and used to narrow the queryset; for example ?skills=django,react&work_from_home=on&min_stipend=5000.

The template iterates over the internships context variable and displays a card for each posting. New postings may be created via the Django admin or by extending the site with a form/API.

Getting Started

Setup steps are identical to those described in the root README: create a virtualenv, install dependencies, run migrations, and start the server.

Refer to README.md in the project root for full developer documentation.
